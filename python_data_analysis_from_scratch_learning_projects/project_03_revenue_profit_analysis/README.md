Project 01 - Sales Performance Analysis

Objective  
Analyze a corporate sales dataset to calculate key financial metrics such as total revenue, total profit, average profit margin, and identify the best-performing products and regions.

Learning Points

- Loading and exploring datasets using pandas
- Handling missing data using groupby and transform
- Converting and sorting datetime columns
- Creating financial metrics (Revenue, Profit, Profit Margin)
- Using groupby to analyze performance by product and region
- Identifying top-performing products using sorting
- Building pivot tables to analyze monthly revenue trends
- Calculating correlation between sales volume and profit
