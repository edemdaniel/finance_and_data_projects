Project 02 - Electronics Sales Analysis

Objective
Analyze sales data in the Electronics category to:
- Calculate total revenue per product
- Determine average revenue per product
- Identify high-value transactions
- Find the top-performing salesperson
- Highlight the single largest transaction

Learning Points
- Filtering data using conditions (`df[condition]`)
- Grouping and aggregating data (`groupby`)
- Calculating revenue and metrics
- Identifying high-value transactions
- Sorting and selecting top records
- Summarizing insights from transaction data
