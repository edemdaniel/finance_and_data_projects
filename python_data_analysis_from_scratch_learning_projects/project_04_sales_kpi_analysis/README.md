Project 04 - Sales KPI Analysis

Objective: Analyze sales transactions across products, categories, and regions to:
- Calculate total revenue per product
- Compute total and average profit per product
- Determine average units sold and profit margin
- Identify top-performing products and managers
- Build KPI tables and pivot tables to summarize insights

Learning Points:

- Filtering data using conditions (df[condition])
- Grouping and aggregating data (groupby and agg)
- Calculating new columns (Revenue, Profit, Profit Margin)
- Merging multiple datasets (sales, products, regions)
- Sorting and ranking top products or managers
- Summarizing insights from sales data using pivot tables
- Basic time-based analysis using Month and Year
